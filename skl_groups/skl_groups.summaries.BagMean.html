skl_groups.summaries.BagMean¶

class skl_groups.summaries.BagMean¶

Transforms a list of features or Features object by just taking the mean of each entry.

Examples

If you have a kernel where you can compute an (approximate?) explicit embedding, you can compute an explicit embedding of the mean map kernel (MMK, also known as maximum mean discrepancy or MMD) by first doing the kernel embedding and then taking the mean.

For example, using an approximate RBF kernel:

get_rbf_mmk = Pipeline([
    ('rbf', sklearn.kernel_approximation.RBFSampler()),
    ('mean', BagMean()),
])
