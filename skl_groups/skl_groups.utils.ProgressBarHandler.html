skl_groups.utils.ProgressBarHandler¶

class skl_groups.utils.ProgressBarHandler(widgets=None, **kwargs)[source]¶

A logging handler that uses the progressbar module to show progress from a ProgressLogger.

Takes the same parameters as progressbar.ProgressBar, but gives a default for widgets that applies only when maxval is available; you’ll need to pass different widgets if not.
