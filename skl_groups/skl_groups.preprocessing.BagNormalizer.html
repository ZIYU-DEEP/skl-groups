skl_groups.preprocessing.BagNormalizer¶

class skl_groups.preprocessing.BagNormalizer(norm='l2')[source]¶

Normalizes each sample individually to have unit norm (l1 or l2).

This is just BagPreprocesser with sklearn.preprocessing.Normalizer.

Parameters:

norm : ‘l1’ or ‘l2’, optional, default ‘l2’

The norm to use to normalize each nonzero sample.
