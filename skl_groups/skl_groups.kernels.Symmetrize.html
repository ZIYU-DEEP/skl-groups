skl_groups.kernels.Symmetrize¶

class skl_groups.kernels.Symmetrize(copy=True)¶

Symmetrizes pairwise affinity/distance/whatever matrices, by taking the mean of itself and its transpose.

Parameters:

copy : boolean, optional, default True

If false, invalidates the passed-in matrix.

Notes

copy=False currently doesn’t do anything.
