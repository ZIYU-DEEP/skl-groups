API reference¶

Features representation¶

features.Features A wrapper class for storing bags of features.

Preprocessing¶

preprocessing.BagMinMaxScaler Linearly scales each feature dimension to lie within the given range, for example [0, 1].
preprocessing.BagNormalizer Normalizes each sample individually to have unit norm (l1 or l2).
preprocessing.BagPCA Runs principal components analysis to reduce the dimensionality of the features.
preprocessing.BagPreprocesser Applies a preprocessing estimator to each column of the bags, independently.
preprocessing.BagStandardizer Standardizes each feature dimension to have zero mean and unit variance, regardless of the bag it falls into.

Summaries¶

Methods that convert set-based features into single feature vectors.

summaries.BagMean Transforms a list of features or Features object by just taking the mean of each entry.
summaries.BagOfWords Transforms feature sets into the bag of words representation:
summaries.L2DensityTransformer Summarizes sets into a single vector, whose inner products estimate the L2 inner product between the density functions underlying each of the sets.

Divergences¶

These methods assume feature sets are samples from some probability distribution and estimate divergences, distances, and so on between those distributions.

divergences.KNNDivergenceEstimator Estimates various divergence functions between bags.

Kernel utilities¶

Utilities to convert generic affinities into valid kernels.

kernels.PairwisePicker Picks a subset of the passed-in matrix.
kernels.Symmetrize Symmetrizes pairwise affinity/distance/whatever matrices, by taking the mean of itself and its transpose.
kernels.RBFize Turns a distance matrix into an RBF kernel: \(K(x, y) = \exp\left( - \gamma \lVert x - y \rVert^2 \right)\).
kernels.ProjectPSD Projects a pairwise square symmetric affinity matrix to be positive semidefinite, by discarding any negative eigenvalues from its spectrum.
kernels.FlipPSD Makes a pairwise symmetric square affinity matrix into a valid positive semidefinite kernel, by flipping the sign of any negative eigenvalues in its spectrum.
kernels.ShiftPSD Makes a pairwise square affinity matrix into a valid positive semidefinite kernel, by flipping the sign of any negative eigenvalues in its spectrum.
kernels.SquarePSD Makes a pairwise symmetric square affinity matrix into a valid positive semidefinite kernel by squaring its eigenvalues (via S -> S S^T).
